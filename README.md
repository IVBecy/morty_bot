# Morty
A fun discord bot

# Usage:
### Music:
- play
    - Plays music from url / keyword `$play [url / keyword]`
- skip / stop
    - Skips current song
- leave
    - Leaves vc
- queue / q
    - Shows music queue

### Fun:
- nsfw
    - Embeds an nsfw GIF
- pee-pee
    - Shows size

# Libs:
`pip install -r requirements.txt`

# Execute:
- `py bot.py`
